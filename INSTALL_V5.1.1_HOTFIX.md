# DeadEnd Scents V5.1.1 Hotfix

This hotfix restores the Apps Script `?action=settings` read endpoint used by the public website to load the live Settings tab.

## Install
1. Open the Google Sheet.
2. Go to Extensions > Apps Script.
3. Replace `Code.gs` with `google-apps-script/Code.gs` from this ZIP.
4. Save.
5. Deploy > Manage deployments > Edit > New version > Deploy.
6. Hard refresh the website and confirm Fragrance of the Week reads the live weekly discount from Settings.

No public site design files are changed by this hotfix.
