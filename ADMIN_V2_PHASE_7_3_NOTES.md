# Admin V2 Phase 7.3 — Inventory Intelligence

Admin-only release. No public storefront files were intentionally changed.

## New
- `admin/inventory-intelligence.html`
- `admin/inventory-intelligence.js`

## Updated
- `admin/admin-shell.js`

## Data
Uses the existing `catalogue` and `orderItems` read endpoints. No Apps Script redeployment or spreadsheet schema change is required.

## Main features
- Stock value and projected remaining revenue
- Bottle health status
- Promote-this-week suggestions
- Reorder candidates
- Slow movers and capital-at-risk estimates
- Do-not-rebuy warnings
- Scent Style stock balance
- Full sortable/filterable bottle scorecard
