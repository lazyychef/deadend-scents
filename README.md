# DeadEnd Scents Website

Cheapest setup: upload these files to a public GitHub repository and enable GitHub Pages.

## Update stock/prices
Edit `data.js`. Each fragrance is one object.

## Go live
1. Create a GitHub account.
2. Create a public repository called `deadend-scents`.
3. Upload all files in this folder.
4. Go to Settings → Pages → Deploy from branch → `main` → root.
5. Your site will publish at `https://YOURUSERNAME.github.io/deadend-scents/`.

## Custom domain later
Buy a domain and set it in GitHub Pages settings.
