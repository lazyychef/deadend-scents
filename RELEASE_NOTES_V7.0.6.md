# DeadEnd Scents V7.0.6

## Release type
Public site UI update only.

## Changed
- Refined the public homepage into the V7 cream / forest green / warm wood design direction.
- Kept the current DeadEnd Scents logo.
- Updated public navigation labels from long legacy wording to shorter shopper labels.
- Tightened the hero area for mobile and desktop.
- Removed the heavy dark/gradient look with flat boutique styling.
- Reworked discovery packs into expanding tiles on mobile and desktop.
- Reduced pack vertical bulk while keeping scent lists accessible on tap/click.
- Cleaned catalogue cards with flatter borders, clearer hierarchy and better price button spacing.
- Tightened the search/filter panel and quick filter buttons.
- Restyled the featured fragrance, cart and order sections to match the V7 public UI.

## Not changed
- Google Sheets integration.
- Apps Script.
- Admin pages or admin logic.
- SEO engine files.
- Inventory logic.
- Catalogue data files.

## Changed files
- `index.html`
- `styles.css`
- `app.js`
- `RELEASE_NOTES_V7.0.6.md`
