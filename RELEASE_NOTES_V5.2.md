# DeadEnd Scents V5.2 — Inventory Manager

## Summary

V5.2 upgrades the admin inventory workflow.

The public website is left alone.

## Added

- New mobile-first Inventory Manager.
- Inventory dashboard stats:
  - bottle count
  - estimated inventory value
  - potential 10mL revenue
  - low stock count
- Bottle cards with image thumbnails, mL remaining, value and badges.
- Full bottle editor:
  - House
  - Fragrance
  - Collection
  - Scent Style
  - Fragrantica
  - Description
  - Purchase Date
  - Purchase Price
  - Bottle Size
  - Current mL
  - RRP
  - Purchase Source
  - Seller
  - Condition
  - 3mL / 5mL / 10mL prices
  - Featured toggle
  - Staff Pick toggle
  - Image URL
  - Private Notes
- Inline price calculator inside the bottle editor.
- Competitor 3mL / 5mL / 10mL fields for market checking.
- Accept suggested prices button.
- Image preview before saving.

## Updated

- Admin dashboard now points users to Inventory Manager first.
- New arrivals now use Added Date / Purchase Date and the live Settings tab value.
- Low stock now uses the live Settings tab threshold.
- Apps Script write-back handles more Catalogue columns by name.

## Not changed

- No public homepage redesign.
- No customer catalogue redesign.
- No SEO page changes.
