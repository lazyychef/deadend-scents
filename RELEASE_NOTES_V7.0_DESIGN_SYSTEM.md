# DeadEnd Scents V7.0 — Design System Branch

## Purpose
Create a separate redesign branch for testing a flatter, cleaner, mobile-first UI without changing the live `main` branch.

## Design direction
- Main colour: Cream `#FFFDD0`
- Accent: Deep Forest Green `#182D09`
- Accent: Warm Wood Tone `#A65B32`
- Headings: Cormorant Garamond
- Body/UI: Inter
- Flat layout: reduced gradients, bubbles, shadows and glass effects
- More minimal catalogue cards
- Product tiles no longer display emoji rows
- Current DeadEnd Scents logo retained
- No AI images added

## Pages covered
- Homepage / catalogue
- Discovery packs
- Cart/order section
- SEO house/fragrance/pack pages through `seo-pages.css`
- Admin dashboard
- Inventory manager
- Add bottle
- Calculator
- Features
- Settings
- SEO Engine

## Notes
This is intended for a GitHub branch such as `redesign-v7`, not direct upload to `main` until tested.
