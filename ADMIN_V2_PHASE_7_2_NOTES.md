# DeadEnd Scents Admin V2 — Phase 7.2

Admin-only Wishlist Intelligence upgrade.

## Added
- Ranked Buy Next / Wait for Deal / Sample First / Low Priority tiers
- Wishlist score, ROI, landed-cost, newest and Scent Style sorting
- Status and Scent Style filters
- Inline Wishlist status updates saved to Google Sheets
- Mark as Bought action
- Side-by-side comparison for up to three bottles
- Catalogue gap summary based on current Scent Style counts
- Duplicate Catalogue / Wishlist warning during analysis
- Add Bottle prefill from a Wishlist item
- Mobile-friendly Wishlist controls

## Deployment
Replace:
- admin/intelligence.html
- admin/intelligence.js
- admin/admin.css
- admin/add-bottle.js

No Apps Script redeployment and no spreadsheet schema change are required.
